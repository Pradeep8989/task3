
# Task 3: Linear Regression

## Objective
To implement simple and multiple linear regression to predict house prices.

## Steps Performed
- Loaded dataset using Pandas
- Converted categorical data using one-hot encoding
- Split data into training and testing sets
- Applied Linear Regression model
- Evaluated using MAE, MSE, and R² score
- Visualized actual vs predicted values

## Evaluation Metrics
- MAE: Measures average error
- MSE: Penalizes large errors
- R² Score: Indicates model accuracy

## Insights
- Model predicts house prices based on multiple features
- Some features have stronger influence (higher coefficients)
- R² score shows how well model fits the data
- Difference between actual and predicted shows prediction error
- Model performance depends on quality of features
